# ClauseWise Leguism ⚖️

ClauseWise Leguism is an NLP-based application that summarizes complex Terms and Conditions documents and presents them in a simplified format for easy understanding. The system uses advanced Natural Language Processing techniques and transformer models to generate concise summaries of legal clauses.

---

## 🚀 Project Overview

Legal documents and terms & conditions are often lengthy and difficult to understand.  
ClauseWise Leguism solves this problem by automatically summarizing legal text and presenting key information in a clear and concise format.

The system leverages **spaCy NLP for text processing**, **T5 Transformer for abstractive summarization**, and **Azure Speech SDK for text-to-speech**, providing both readable and audible summaries.

---

## 🛠️ Tech Stack

- **Programming Language:** Python  
- **NLP Framework:** spaCy  
- **Deep Learning Model:** Fine-tuned T5 Transformer  
- **Speech Integration:** Microsoft Azure Speech SDK  
- **Frontend / Deployment:** Streamlit  

---

## ✨ Features

- 📄 Automatic summarization of complex Terms & Conditions
- 🤖 Abstractive text summarization using fine-tuned T5 model
- 🧠 NLP preprocessing using spaCy
- 🔊 Text-to-speech conversion using Azure Speech SDK
- 🌐 Interactive web application using Streamlit
